Research

As an aspiring researcher, my goal is to develop data-driven solutions that support sustainable development and make machine learning models more understandable, reliable, and trustworthy. One of my contributions is a method for detecting adulterated oil using machine learning, helping to create transparent and dependable AI systems for real-world use.

In addition to my research, I have led several practical projects. At the Federal University Lokoja, I developed a machine learning system to predict GPA, with a focus on clearly explaining the factors that affect student performance. I also created a deep learning-based road fault detection system that accurately identifies infrastructure issues, prioritizing reliability and ease of use for maintenance.

Publications

  • A computer-vision based system for detecting road anomalies in Nigeria. : Ahmad Khalid*, Temitope Oluwafemi. (Manuscript under preparation.)

Completed Projects

  • Road Anomaly Detection System: Developed a machine learning model to detect road surface anomalies (e.g., potholes, cracks) using image data with CNNs, deployed with Flask.
  • GPA Prediction System: Built a machine learning model to predict student GPAs based on academic data, using Linear Regression, Decision Trees, and Random Forest models.

Ongoing Project

  • Intelligent System for the Classification of West African Currency: Developing an AI-based system using computer vision and machine learning to accurately identify and classify the country of origin based on West African currencies, with the goal of facilitating trade, tourism, and economic activities between countries in the region, while reducing the risk of errors in currency exchange..
